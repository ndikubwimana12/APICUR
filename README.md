# APICUR TSS - School Management System

A comprehensive web-based School Management System designed for **APICUR Technical Secondary School** to streamline academic and administrative operations.

### ğŸ¯ Role-Based Access Control
The system supports multiple user roles with specific functionalities:

#### 1. **Admin**
- Full system administration
- Manage users (create, edit, delete)
- Manage classes, subjects, and academic structure
- View reports and analytics
- Monitor activity logs
- Configure system settings

#### 2. **Secretary**
- Register and manage students
- Upload and manage meeting documents
- Maintain student records
- Assign students to classes

#### 3. **Teacher**
- View assigned classes and students
- Record student attendance
- Enter marks (Formative, Continuous Assessment, Exam)
- Generate term report forms
- Upload pedagogical materials
- View timetables assigned by DOS

#### 4. **Director of Studies (DOS)**
- Create and manage timetables
- Assign teachers to classes and subjects
- Monitor teacher performance
- View reports and academic progress

#### 5. **Head Teacher**
- Oversee all school operations
- Review performance reports
- Manage discipline and staff
- Approve report cards and results

#### 6. **Accountant**
- Manage school fees
- Record payments and generate receipts
- Track outstanding balances
- Generate financial reports

#### 7. **Discipline Officer**
- Record student discipline cases
- Manage behavior reports
- Notify parents
- Generate discipline summaries
